어떠한 뛰어난 해커가 북한 정부의 비밀을 파헤치기 위해 평양시의 감시 카메라 N개를 해킹하였다. 평양시는 1000 x 1000 크기의 2차원 격자 판으로 모델링된다. i번째 감시 카메라는 x1,i ≤ x ≤ x2,i, y1,i ≤ y ≤ y2,i 인 정수 격자점 (x, y)을 촬영할 수 있다.
평양시의 감시 카메라의 해상도는 좋지 않아서, 어떠한 격자점을 감시하기 위해서는 3개의 감시 카메라가 해당 격자점을 동시에 촬영할 수 있어야 한다. 해커는 N개의 감시 카메라 중 정확히 3개의 카메라를 해킹하여서, 각 카메라가 동시에 감시할 수 있는 격자점이 존재하도록 하려고 한다.
해커가 이 조건을 만족하면서 카메라 3개를 고를 수 있는 경우의 수를 계산하여라.

[입력]
첫 번째 줄에 테스트 케이스의 수 TC가 주어진다. 이후 TC개의 테스트 케이스가 새 줄로 구분되어 주어진다. 각 테스트 케이스는 다음과 같이 구성되었다.
  ∙ 첫 번째 줄에 감시 카메라의 개수 N 이 주어진다. (3 ≤ N ≤ 100000)
  ∙ 이후 N개의 줄에, 감시 카메라가 촬영할 수 있는 영역을 나타내는 4개의 정수 x1,i, y1,i, x2,i, y2,i가 주어진다. (1 ≤ x1,i ≤ x2,i ≤ 1000, 1 ≤ y1,i ≤ y2,i ≤ 1000)

[출력]
각 테스트 케이스 마다 한 줄씩, 문제의 정답을 출력하라.
